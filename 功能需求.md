前端：Vue

5.切换视频(上一个，下一个)  
6.退出播放器  
7.视频收藏(？)  
8.亮度调节  
9.视频转向(？)  
10.快进And快退

一、项目架构  
本项目采用前后端分离结构，主要由三大模块构成：

1. 后端（Flask）
视频播放状态管理（播放、暂停、进度、音量等）  
手势识别服务（基于 Mediapipe 和 OpenCV）  
用户数据存储（收藏、播放历史、模式等）  
视频资源管理与路由接口      

2. 前端（Vue + Element UI / Vant）  
视频播放 UI（可使用 \<video> 标签或封装组件）  
手势功能提示与开关  
视频控制栏（进度、音量、播放模式等）  
播放列表与收藏列表视图  
响应后端提供的控制命令

3. 手势识别模块（摄像头接入 + 手势识别）  
使用 Mediapipe 获取手部关键点  
使用 OpenCV 接收摄像头视频流  
映射手势到控制指令（与播放器联动）  
可视化调试模式（展示识别手势与动作）
